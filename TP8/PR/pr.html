Abstract

No trabalho de casa 8, foi pedido para terminar o desenvolvimento de uma API, que fosse capaz de
comunicar com o GraphDB, para o transporte de informação acerca da ontologia de cinema.
Neste caso, foram desenvolvidos os GETs necessários para os dados dos filmes, atores e personagens
presentes na base de dados.
