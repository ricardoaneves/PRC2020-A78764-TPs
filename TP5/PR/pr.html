Abstract

Este trabalho consistiu no desenvolvimento de uma query SPARQL, fazendo uso da interface do website https://dbpedia.org/sparql.
Esta query acede a todos os registos de nadadores (Swimmers) presentes na topologia, apresentando o seu nome, género,
data e local de nascimento, data de morte e abstract, sendo que alguns destes parâmetros são opcionais e podem não estar contidos
na topologia fornecida.
Os resultados obtidos encontram-se em formato .JSON.
