Abstract

Este segundo trabalho consistiu na criação de uma ontologia que relacioná-se alguns elementos presentes
no XML dado pelo professor, que armazena um grande arquivo de músicas digital.
Em primeiro lugar, foi criado um script em Python de modo a ler o ficheiro XML fornecido, e traduzir o seu
conteúdo para formato .TTL. Deste modo, foi criado um novo ficheiro dados.ttl
onde está armazenada toda a informação das várias obras, instrumentos e partituras.
De seguida, com a ferramenta Protegé, foi criada a base da ontologia, identificando as classes, as suas propriedades e
as ligaçoes entre as mesmas.
Assim, copiei o conteúdo do ficheiro gerado (dados.ttl) para o ficheiro .TTL que contém toda a ontologia, ficando assim completa.
Numa segunda parte do trabalho, foram criadas quatro queries para dar resposta às questões fornecidas. Estas queries podem ser
consultadas no ficheiro queries.txt, como podem ver na secção deliverables.
