Abstract

Este primeiro trabalho consistiu na criação de uma ontologia que relacionasse os vários elementos
do Mestrado de PLC, tendo como base uma ontologia básica criada préviamente na aula de PRC.
Em primeiro lugar, foi criado um script em Python de modo a ler as várias linhas de um ficheiro .CSV,
em que cada uma corresponde à informação de um único aluno. Deste modo, foi criado um novo ficheiro .TTL,
onde está armazenada toda a informação dos vários alunos.
De registar que as Unidades Curriculares e respetivos docentes foram criados manualmente na ferramenta Protegé,
devido a tratarem-se de dados muito reduzidos em tamanho.
Assim, copiei o conteúdo do ficheiro gerado (alunos.ttl) para o ficheiro .TTL que contém toda a ontologia, ficando assim completa.
