Abstract

Este terceiro trabalho consistiu na criação de um servidor NodeJS para alocar uma interface onde
é possível escolher um repositório e aplicar queries SPARQL diretamente sobre ele.
aAssim, com base no exemplo disponibilizado pelo docente durante a aula, foi desenvolvido um servidor
e a sua interface, de modo a ser utilizado pelo utilizador comum.
Os pedidos para o GraphDB são efetuados a partir de pedidos axios, já bastante conhecidos do semestre passado.
